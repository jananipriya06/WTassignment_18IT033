Transpose of a matrix in single line in python

Python provides interesting way to find the transpose of a matrix using zip() method . Matrix is represented as a nested list. Zip returns an iterator of tuples, where the i-th tuple contains the i-th element from each of the argument sequences or iterables.

Example:
matrix = [[1,2],[3,4],[5,6]] t_matrix = zip(*matrix)

In this example we unzip our array using * and then zip it to get the transpose.
t_matrix is a tuple ((1,3,5),(2,4,6))
Reference : https://www.geeksforgeeks.org/transpose-matrix-single-line-python/
